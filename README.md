# Tech-Sales-Dashboard
This project is a high-performance Tech Sales Dashboard built with React JS, Vite, Chart.js, React Icons, HTML5, and CSS3.
It delivers a sleek, modern interface featuring interactive charts, responsive layouts, and an optimized user experience.
Designed for visualizing fixed sample sales data, this dashboard provides a clean and professional look suitable for demos, analytics previews, and UI showcases.
